# Web Portal E-Statement

This repository is a portfolio showcase of my internship project developed in a real banking environment. Due to confidentiality and internal organizational restrictions, the source code, internal system screenshots, and sensitive implementation details are not publicly shared.

## Overview
Web Portal E-Statement is an internal web application designed to help central banking staff request customer statements for accounts with high transaction volumes. The system supports statement request submission, PDF generation, email delivery, and downloadable files through the web portal.

## My Role
- Analyzed internal workflows and service processes in a banking environment
- Participated in discussions, workflow analysis, and system planning
- Contributed to frontend and backend development
- Supported system integration involving database connectivity, PDF generation, file handling, and email delivery
- Assisted with testing, UAT, deployment, and internal rollout

## Technologies Used
- Vue.js
- Spring Boot
- MySQL
- JasperReports
- RESTful API
- Git

## Key Features
- Internal login for authorized staff
- Online E-Statement request form
- PDF statement generation
- Email delivery to recipients
- File download through the web portal
- Request history tracking
